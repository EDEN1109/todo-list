# TODO List

일상의 할 일을 깔끔하게 정리할 수 있는 심플한 웹 앱입니다.  
별도 설치나 서버 없이 브라우저에서 바로 사용할 수 있습니다.

---

## 주요 기능

### 할 일 관리
- 할 일 추가 / 완료 체크 / 삭제
- 완료 시 완료 시각을 `YYYY-MM-DD HH:mm` 형식으로 표시
- 완료 항목 일괄 삭제

### 설명(Description)
- 할 일 작성 시 선택적으로 설명을 추가할 수 있습니다
- 목록에서 ▼ 버튼을 눌러 설명을 접고 펼칠 수 있습니다

### 그룹
- 자유롭게 그룹을 만들고 삭제할 수 있습니다 (예: Home, Work, 공부 등)
- 할 일을 작성할 때 그룹을 지정할 수 있습니다
- 그룹 삭제 시 해당 그룹의 할 일은 미분류 상태로 전환됩니다

### 필터
- **그룹 필터**: 전체 또는 특정 그룹의 할 일만 볼 수 있습니다
- **상태 필터**: 전체 / 진행 중 / 완료로 나눠서 볼 수 있습니다
- 두 필터는 독립적으로 조합할 수 있습니다

---

## 시작하기

별도 설치가 필요 없습니다. `index.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.

```
todo-list-main/
├── index.html   # 앱 진입점
├── style.css    # 스타일
└── app.js       # 앱 로직
```

## 데이터 저장

현재 데이터는 브라우저 메모리에만 저장됩니다.  
페이지를 새로고침하거나 탭을 닫으면 입력한 내용이 초기화됩니다.
